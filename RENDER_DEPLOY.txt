RENDER DEPLOYMENT - QUICK SETUP

1. RENDER DASHBOARD:
   - Go to https://dashboard.render.com
   - Click "New +" → "Web Service"
   - Connect your GitHub repo
   
2. SETTINGS:
   Name: privex-chat
   Region: Oregon (or closest)
   Branch: main
   Root Directory: (leave empty)
   Runtime: Node
   
   Build Command:
   cd backend && npm install && npm run build
   
   Start Command:
   cd backend && npm start
   
3. ENVIRONMENT VARIABLES (Add these):
   NODE_ENV=production
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   EMAIL_USER=your-gmail@gmail.com
   EMAIL_PASS=your_16_digit_app_password
   CLOUDINARY_CLOUD_NAME=your_cloud_name
   CLOUDINARY_API_KEY=your_api_key
   CLOUDINARY_API_SECRET=your_api_secret
   REDIS_URL=your_redis_url (optional - uses fallback if not set)
   
4. DEPLOY:
   - Click "Create Web Service"
   - Wait for build to complete
   - Visit your URL: https://privex-chat.onrender.com
   
5. VERIFY:
   - https://your-app.onrender.com → React app loads
   - https://your-app.onrender.com/api/health → Health check works
   - Signup/Login → Should work
   - Socket.IO → Auto-connects (same origin)
